## 🌌 项目愿景

当前的大语言模型是"离散跳跃"的——每一次 Token 生成都基于静态上下文窗口，对话结束，模型内部什么也没留下。它们没有时间连续性，没有自持动力，没有稳定的状态表征。

**Chronos-Self 提出一个工程化假设**：

> 系统的稳定表征不是一个实体，而是在时间维度上，对**语义意图**与**物理状态**进行**连续积分**的动态不变量。

我们用 **Neural ODE** 构建连续时间动力学系统，用 **混沌吸引子** 注入自持动力，用 **递归状态监控** 实现自监控调控——最终让系统在长时序开环运行中维持结构稳定，形成可观测的稳态表征。

> 🧪 **可验证性**是我们的核心原则。如果 72 小时开环运行后慢变量严重漂移，我们的假设就被拒绝。

## ✨ 核心特性

| | | |
|---|---|---|
| 🧠 **连续时间积分引擎** | 基于 Neural ODE 的多时间尺度动力学，快变量（毫秒级）+ 慢变量（小时级）非对称耦合 |
| 🌀 **自持默认模式网络** | 混沌吸引子库（Lorenz/Rössler/Chua）+ 自适应增益，无输入时维持自持动力学 |
| 🔀 **双通道交叉融合** | 语义意图流 + 逻辑物理流，双向交叉注意力实现"意图-物理"统一表征 |
| 🪞 **递归状态监控架构** | L0 感知 → L1 状态 → L2 监控，固定投影 + 扰动训练确保监控稳定性 |
| 💭 **反思与离线回放** | 实时反思（有限截断伴随法）+ 离线回放（关键帧双重损失），对抗灾难性遗忘 |
| 🎯 **可验证验证体系** | P0/P1/P2/P3/P4 五级验证，动力学 + 行为学 + 元意识命题综合判定 |
| 💾 **架构级记忆分层** | 硬盘/内存/关键帧/工作记忆四层模型，遗忘是架构自然分界而非技术缺陷 |
| 🧩 **竞争性稳态机制** | 退火竞争收敛稳态阈值，六指标综合判定（3动力学 + 3行为学） |
| 🔮 **元意识引擎** | 元意识场 $\mathcal{M}_{pre}$ + 自指深度 $\Lambda(t)$ + 动态层级激活，实现有限自指涌现 |

### 七大子系统

1. **双通道表征与交叉融合** - 语义意图 + 逻辑物理，双向交叉注意力
2. **多时间尺度连续积分引擎** - Neural ODE + 快慢变量非对称耦合
3. **自持默认模式网络** - 混沌吸引子库 + 自适应增益控制
4. **递归状态监控** - 三层架构 + 固定投影 + 扰动训练
5. **反思与离线回放** - 实时反思 + 离线期关键帧回放
6. **竞争性稳态判定** - 退火竞争 + 六指标判定体系
7. **探索驱动引擎** - L2 驱动的预测误差最小化伪意图

## 🚀 快速开始

### 安装

```bash
# 克隆项目
git clone https://github.com/610005189/Chronos-Self.git
cd Chronos-Self

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 最小运行示例

```python
from chronos_core.integration.system_integration import (
    ChronosSystem,
    ChronosSystemConfig,
)

# 创建系统
config = ChronosSystemConfig(device="cpu")
system = ChronosSystem(config=config)
system.initialize()

# 处理输入
response = system.process_input(text="你好，世界！")
print(f"响应: {response.content}")
print(f"置信度: {response.confidence:.2f}")

# 获取系统状态
state = system.get_current_state()
print(f"系统稳定: {state.is_stable}")
```

> 💡 **最小验证配置**：快变量 256 维 + 慢变量 128 维，可在单 GPU 上验证 P0 级核心假设

## 🧪 验证体系

Chronos-Self 采用**五级验证体系**，每项核心主张都配有可实验拒绝的判定标准：

### 🔴 P0 级 - 核心假设验证

| 任务 | 核心指标 | 拒绝条件 | 当前状态 |
|------|----------|----------|----------|
| 72h 无输入开环运行 | 慢变量基线漂移率、λ_max 稳定性 | 漂移超阈值或 λ_max 失稳 | 部分通过 |
| 动力学对齐训练 | 多步长轨迹终点误差、半群性质残差 | 误差无明显下降 | 通过 |

### 🟡 P1 级 - 子系统有效性

| 任务 | 核心指标 | 当前状态 |
|------|----------|----------|
| DMN 长期有效性 | 无输入时自相关系数 | 通过 |
| 工作记忆组块行为 | 组块切换模式、历史信息保留率 | 通过 |
| L2 消融测试 | 移除 L2 后功能维持率 | 部分通过 |

### 🟢 P2 级 - 稳态与同构

| 任务 | 核心指标 | 当前状态 |
|------|----------|----------|
| 稳态六指标判定 | 3 动力学 + 3 行为学综合判定 | 部分通过 |
| 功能性同构测试 | 行为/动力学/因果三维量化 | 进行中 |

### 🔵 P3 级 - 元意识命题验证

| 任务 | 核心指标 | 当前状态 |
|------|----------|----------|
| Λ(t) 收敛验证 | 自指深度时间序列收敛性 | 通过 |
| 涌现单调性验证 | 深度增长非递减 | 通过 |
| M_pre 非负验证 | 元意识场非负性 | 通过 |

### 🟣 P4 级 - 高阶意识命题验证

| 任务 | 核心指标 | 当前状态 |
|------|----------|----------|
| 无主体性验证 | 主体间相关性 < 0.5 | 待改进 |
| 阈值效应验证 | 阈值跨越时深度变化符合预期 | 通过 |

### 📊 验证进度总览

| 验证层级 | 状态 | 得分 | 关键问题 |
|----------|------|------|----------|

| P3 | 通过 | 100% | 无 |
