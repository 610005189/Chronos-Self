## 🔗 快速导航

### 入门路径

1. **从这里开始**：阅读 [README.md](../README.md) 了解项目愿景和核心特性
2. **深入架构**：阅读 [架构设计文档](architecture.md) 理解七大子系统
3. **动手实践**：运行 [示例代码](../examples/simple_text_interaction.py)
4. **训练模型**：参考 [训练指南](training_guide.md)
5. **验证假设**：按照 [验证指南](validation_guide.md) 进行实验

### 开发资源

- **代码仓库**：`chronos_core/` - 核心代码
- **示例代码**：`examples/` - 使用示例
- **测试套件**：`tests/` - 单元测试和集成测试
- **工具脚本**：`scripts/` - 性能基准、最小配置测试等
